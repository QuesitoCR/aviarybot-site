Esta Política de Privacidad explica cómo AviaryBOT (el “Bot”) puede recopilar y usar datos cuando interactúas con él en Discord. El Bot fue creado por xMax (~unstudiomax~) para la comunidad de Jai Aviary.

Resumen rápido: el Bot solo usa datos necesarios para funcionar y moderar; no vende datos.

1. Datos que el Bot puede procesar

Dependiendo de las funciones habilitadas, el Bot puede procesar:

  • IDs de Discord (usuario, servidor, canal, mensaje) para identificar acciones y configuraciones.
  • Contenido de comandos (por ejemplo, mensajes que empiezan con un comando) para ejecutar la acción solicitada.
  • Configuraciones del servidor relacionadas a funciones del Bot (por ejemplo: canales permitidos, roles, ajustes).

2. Lo que el Bot NO hace

  • No vende ni alquila datos.
  • No solicita contraseñas.
  • No intenta leer mensajes privados (DMs) a menos que tú le escribas directamente y sea necesario para una función específica.

3. Finalidad del uso de datos

  • Operar comandos y funciones del Bot.
  • Moderación, anti-spam y seguridad del servidor (si está habilitado).
  • Guardar preferencias y configuración para que el Bot funcione correctamente.
  • Diagnóstico básico de errores (logs) para mantener estabilidad y mejoras.

4. Retención de datos

Los datos se conservan solo el tiempo necesario para operar el Bot y mantener configuraciones. Cuando el Bot es removido de un servidor, las configuraciones asociadas pueden eliminarse dentro de un plazo razonable.

5. Compartición de datos

Los datos no se comparten con terceros salvo:

  • Proveedores técnicos necesarios para operar el Bot (por ejemplo, hosting), solo en la medida requerida.
  • Requerimientos legales válidos, si aplicara.

6. Seguridad

Se aplican medidas razonables para proteger datos (por ejemplo, manejo seguro de tokens y controles de acceso). Aun así, ningún sistema es 100% infalible.

7. Tus derechos y solicitudes

Si quieres solicitar eliminación de datos relacionados a tu usuario o a un servidor, o tienes dudas, contáctame desde: unstudiomax.carrd.co

8. Cambios a esta política

Esta política puede actualizarse. La fecha de “Última actualización” indica el cambio más reciente.
